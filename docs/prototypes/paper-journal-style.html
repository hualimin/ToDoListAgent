为什么是「纸本日记」

待办要快记、反思要慢想、学习要沉读——三种节奏共用一种「沉静纸感」最统一。朱砂红只用在「进行中 / 紧急 / 主行动」上，强引导视觉焦点。

状态如何表达

用墨色细线 + 小色点 + 印章式标签，而不是大色块。紧急=朱砂、延期=琥珀、完成=黛绿（删除线），克制不花。

字体

霞鹜文楷（中文楷体，手账感）+ Newsreader（拉丁衬线，数字日期更精致）。刻意不用 Inter 等通用字。

这是风格原型，非最终功能稿。看方向对不对，再按你的 5 个原型合稿。
